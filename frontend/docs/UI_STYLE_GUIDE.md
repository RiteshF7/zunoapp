# Zuno UI Style Guide

Complete design system and UI guidelines. See [ui/](../ui/) and [landing-ui/](../landing-ui/).

- **App (ui/)**: CSS variables in `ui/src/styles/theme.css`; Tailwind config in `ui/index.html`
- **Landing (landing-ui/)**: Tailwind in `landing-ui/index.html`

Design tokens: colors (background-light, background-dark, card-dark, accent-blue), typography (Inter), border radius (1.25rem, 1.5rem, 2rem), dark mode via Tailwind `dark:` prefix.
